# 📄 ResumeIQ AI — Automated Resume Screening Tool

ResumeIQ AI is an AI-powered Automated Resume Screening & ATS Optimization Platform designed to help recruiters and HR teams intelligently filter, rank, and shortlist candidates based on job descriptions, skills, and resume relevance.

This project simulates a real-world Applicant Tracking System (ATS) using Python, NLP, TF-IDF, cosine similarity, resume parsing, and interactive analytics dashboards.

---

# 🚀 Project Overview

Recruiters often receive hundreds of resumes for a single role.

Manual screening:
- consumes time
- creates inconsistency
- increases hiring delays

ResumeIQ AI automates the screening workflow by:
- extracting resume text
- matching skills with job requirements
- calculating ATS scores
- ranking candidates
- generating shortlist decisions
